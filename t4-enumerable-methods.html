When I was working on a particular problem, I needed to separate things accoridng to groups. This is where I found the Enumerable#group_by method. Let's take a look at what that is.

In simple terms, Enumerable#group_by breaks a collection into groups based on criteria. When using this method, you would first place your parameters in a code block. Ruby will then process this information and return a hash that represents the group. This is done through iteration. Hashes have keys and corresponding values and will make retrieveing information from the hash easier.
